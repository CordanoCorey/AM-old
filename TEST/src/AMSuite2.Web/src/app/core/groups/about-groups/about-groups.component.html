About Groups: Use groups to organize your users to make it easier to administer your account. Groups can be added to meetings to control who can view and modify a meeting. Groups can be administered on a per-member basis on the Members page or per-group on this page. Certain roles can be applied to a group and are explained below:

Group Administrator: The owner of the group. This user is the only one allowed to delete the group along with Account Administrators.

Manager: Can do anything within an assigned group, except delete it. They have access to this page.

Contributor: Allowed to help manage meetings and agendas for this group.

Member: View only access to meetings and agendas of this group.

Voter: Allowed to vote (if enabled for meeting) for this group.

Vote Taker Can take votes (if enabled for meeting) for this group. Use this if you want one user to take votes for everyone else instead of allowing each individual to vote. This is a default setting but can be overridden on a meeting.

Minute Taker Can take minutes in meetings for this group. This is a default setting but can be overridden on a meeting.

Attendance Taker Can take attendance in meetings for this group. This is a default setting but can be overridden on a meeting
